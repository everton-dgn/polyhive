# @evertondgn/polyhive-relay

End-to-end-encrypted relay used by the PolyHive daemon to bridge clients across networks. Internal package — installed automatically as a dependency of `@evertondgn/polyhive-server` and `@evertondgn/polyhive-cli`.

You normally do not depend on this package directly. To use PolyHive, install the CLI:

```bash
npm install -g @evertondgn/polyhive-cli
```

For the protocol design and threat model, see [SECURITY.md in the main repo](https://github.com/everton-dgn/polyhive/blob/main/SECURITY.md).
