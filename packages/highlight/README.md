# @everton-dgn/polyhive-highlight

Lightweight syntax highlighter used by PolyHive's app and server for diffs and code blocks. Wraps Lezer parsers behind a small, deterministic token API. Internal package — installed automatically as a transitive dependency of `@everton-dgn/polyhive-cli`.

You normally do not depend on this package directly. To use PolyHive, install the CLI:

```bash
npm install -g @everton-dgn/polyhive-cli
```

For the main project, see [github.com/everton-dgn/polyhive](https://github.com/everton-dgn/polyhive).
